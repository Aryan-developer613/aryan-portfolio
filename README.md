# Aryan Jaiswal Portfolio

Static portfolio website built with HTML, CSS, and JavaScript.

## Deploy With GitHub Pages

1. Create a new GitHub repository. Recommended name: `aryan-portfolio`.
2. Upload these files to the repository:
   - `index.html`
   - `Aryan_Jaiswal_Resume_Compact.pdf`
   - `README.md`
3. Open the repository on GitHub.
4. Go to `Settings` > `Pages`.
5. Under `Build and deployment`, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.

Your live website link depends on your exact repository name:

```text
https://aryan-developer613.github.io/YOUR-REPOSITORY-NAME/
```

Example, if your repository is named `aryan-portfolio`:

```text
https://aryan-developer613.github.io/aryan-portfolio/
```

If you get a 404, check that:

- `index.html` is uploaded directly in the repo root, not inside a folder or ZIP.
- GitHub Pages source is set to `main` and `/root`.
- The repository name in the link exactly matches your GitHub repository name.
- You waited 1-3 minutes after saving Pages settings.

GitHub Pages usually takes 1-3 minutes to publish after setup.
